# Datasets Used

## Free Datasets
- [MySQL Sample Databases](https://dev.mysql.com/doc/index-other.html) — sakila, world, employees
- [Kaggle Datasets](https://www.kaggle.com/datasets) — e-commerce, HR, retail
- [Mode SQL Tutorial](https://mode.com/sql-tutorial/) — real business data
- [SQLZoo](https://sqlzoo.net) — built-in practice datasets

## By Phase
- Phase 1: sakila (DVD rental), world (countries/cities)
- Phase 2: E-commerce orders, HR/Employee datasets
- Phase 3: SaaS metrics, Fintech transactions
- Phase 4: FAANG-style mock datasets from StrataScratch
